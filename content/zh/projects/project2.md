## 项目概述

承接公司传统 DI 客服智能化替换项目，负责全新大模型 Agent 客服体系的从 0 到 1 核心设计工作。

## 我的角色

AI 产品经理 — 全案方案设计、意图拆解、规范定义

## 核心方案

1. **架构升级**: 区别于传统固定工作流，搭建模块化智能体架构，以提升用户沟通体验、提高客户留资转化为核心目标
2. **意图域拆解**: 完成全量用户 Query 场景拆解，枚举业务全场景意图域，划分不同意图的应答边界、工具调用权限，规避智能体乱应答、越权应答问题
3. **规范体系**: 遵循标准化 Agent 开发规范，拆分模块化文档体系，独立完成角色职责文档、工具调用规范文档的内容梳理与落地定义

## 成果

- 完成 Agent 客服从 0 到 1 的完整方案设计
- 建立规范化的话术应答体系
- 替代老旧规则化客服系统
